# Wireshark Network Traffic Analysis  
## Descripción general

Este proyecto documenta el análisis de tráfico de red utilizando Wireshark en un entorno de laboratorio, con un enfoque orientado a detección, análisis y comprensión de comportamientos normales y anómalos desde la perspectiva de un Blue Team / SOC junior.

El objetivo no es solo capturar paquetes, sino interpretar el tráfico, entender los protocolos involucrados y extraer conclusiones útiles para seguridad.

 ## Objetivos del proyecto

- Comprender el funcionamiento del tráfico de red a nivel paquete

- Analizar protocolos comunes desde una perspectiva de seguridad

- Detectar patrones normales y potencialmente anómalos

- Practicar análisis técnico y documentación de hallazgos

## Entorno del laboratorio

- Sistema operativo: Linux / Windows (laboratorio)

- Herramienta principal: Wireshark

- Tráfico generado de forma controlada

## Herramientas y tecnologías

- Wireshark

- Protocolos analizados:

    - DNS

    - HTTP

    - HTTPS / TLS

    - ICMP

## Casos de análisis realizados
1. Análisis de tráfico DNS

2. Análisis de tráfico HTTP

3. TLS / HTTPS

4. Latencia y conectividad

## Metodología de análisis

Para cada captura se siguió el siguiente proceso:

- Definición del objetivo de análisis

- Captura controlada del tráfico

- Aplicación de filtros

- Identificación de patrones

- Documentación de hallazgos
